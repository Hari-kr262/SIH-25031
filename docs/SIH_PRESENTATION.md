# CivicResolve — SIH 2025 Presentation

## Problem Statement SIH25031
**Government of Jharkhand | Clean & Green Technology Theme**

> Build a crowdsourced platform where citizens report civic issues and government resolves them with full accountability.

---

## The Problem

- Citizens have no easy way to report civic issues
- No transparency in issue resolution
- No accountability for government departments
- No data-driven approach to prioritize issues
- Citizens lose faith in the system

---

## Our Solution: CivicResolve

A complete **Python-based** civic issue management system with:
- 📱 Mobile-friendly cross-platform UI (Flet)
- 🤖 AI-powered issue classification and fraud detection
- 🏆 Gamification to drive citizen engagement
- 📊 Real-time analytics for better governance
- 🔔 Multi-channel notifications (in-app, email, SMS)

---

## Key Features

| Feature | Impact |
|---------|--------|
| AI Categorization | 95% routing accuracy |
| Duplicate Detection | 30% reduction in duplicate reports |
| Gamification | 3x citizen engagement |
| SLA Monitoring | 40% faster resolution |
| Geographic Heatmaps | Proactive issue prevention |
| Voice Reporting | Accessible to all citizens |

---

## Tech Stack (100% Python)

| Layer | Technology |
|-------|-----------|
| API | FastAPI + Uvicorn |
| Database | PostgreSQL + SQLAlchemy |
| Frontend | Flet (Material Design 3) |
| AI/ML | TensorFlow + scikit-learn + OpenAI |
| Background | Celery + Redis |
| Deploy | Docker + docker-compose |

---

## User Roles & Impact

1. **Citizens** — Report, vote, verify, earn rewards
2. **Volunteers** — Ground verification with 2x vote weight
3. **Field Workers** — Fix with geo-verified proof
4. **Dept Heads** — Manage teams, track budget
5. **Municipal Admin** — City-wide oversight
6. **Super Admin** — System governance
